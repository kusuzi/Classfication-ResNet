# Classfication-ResNet
Using RESNET model for transfer learning to classify data sets on Pytorch
##### 文件信息

  |—— classfication-ResNet 
  |———— data                  （下载的cifar数据集，也可以是其他数据集）    
  |———— test_images    （测试图片）    
  |———— weights           （权重文件：-pre表示与训练文件）  
  |————model              （ResNet的网络结构）  
  |————Predict_cifar      (测试文件)  
  |————Train_Cifar        (训练文件)
